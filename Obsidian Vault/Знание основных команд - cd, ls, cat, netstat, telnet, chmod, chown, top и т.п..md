								cd:

Команда cd (change directory) - в Linux используется для смены текущей рабочей директории в терминале. Она позволяет перемещаться по файловой системе, переходить в папки, возвращаться назад или переходить в домашний каталог. 

Основные примеры использования:
* cd /home/jorj/Documents - перейти по абсолютному пути
* cd Documents - перейти в папку внутри текущей директории (относительный путь)
* cd . . перейти на уровень выше (в родительскую папку)
* cd или cd ~ - вернуться в домашний каталог 
* cd - вернуться в предыдущую рабочую директорию

								ls:

Команда ls (list) - предназначена для вывода списка файлов и подкаталогов в указанной директории. Если аргументы не заданы, отображается содержимое текущего каталога. Она позволяет просматривать скрытые файлы, права доступа, размеры и время изменения файлов с помощью различных опций

* ls Простой список имен файлов и папок (без скрытых)
* ls -l Подробный список (права, владелец, размер, дата)
* ls -a Показать все файлы, включая скрытые 
* ls -la Подробный список всех файлов, включая скрытые
* ls -lh Подробный список с размерами файлов в человекочитаемом формате
* ls -R Рекурсивный вывод: отображает содержимое текущей папки и всех подпапок
* ls -t Сортировка по времени изменения (сначала новые)
* ls /path/to/dir Просмотр содержимого по определенной директории

								cat:

Команда cat (concatenate) - используется для чтения, создания и конкатенации файлов. Основная функция - вывод содержимого файлов в терминал, а также объединения нескольких файлов в один. Часто используется для быстрого просмотра небольших текстовых файлов

* cat file.txt выводит текст файла в терминал
* cat file1.txt file2.txt выводит содержимое файлов последовательно
* cat file1.txt file2.txt > combined.txt создает новый файл, объединяя содержимое
* cat > new.file позволяет вводить текст, который будет записан в файл
* cat -n file.txt отображает файл с нумерацией всех строк
* cat -e file.txt показывает символ $ в конце каждой строки

Команда не редактирует файл, а служит только открывает его для чтения и вывода.

								netstat:

netstat (network statistic) - утилита для диагностики сети, отображающая активные соединения (TCP/UDP), таблицы маршрутизации, статистику интерфейсов и открытые порты. Хотя она считается устаревшей (рекомендуется ss), netstat остается популярной для быстрого анализа открытых портов и идентификации процессов. 

* netstat -a показать все соединения (и активные, и слушающие)
* netstat -t показать только TCP-соединения
* netstat -u показать только UDP-соединения
* netstat -l показать только слушающие порты
* netstat -n отображать числовые адреса (IP и порты) вместо символьных имен
* netstat -p показать PID и имя программы, использующей сокет (требует sudo)
* netstat -r показать таблицу маршрутизации 
* netstat -i показать статистику сетевых интерфейсов

								telnet:

Команда telnet - это сетевой инструмент для подключения к удаленным узлам по протоколу TCP. Она используется для администрирования, но чаще - для диагностики открытых портов, тестирования служб (SMTP, HTTP) и отладки, так как не шифрует трафик. Основной синтаксис 
	telnet хост порт

**Проверка открытого порта**:
* telnet 192.168.1.1 80 
* telnet google.com 443*

**Подключение к удаленному серверу (устаревший способ):**
* telnet 192.168.1.10

**Взаимодействие со службами:**
* telnet mail.ex.com 25

						chmod: (моя любимая)

Команда chmod (change mode) используется для изменения прав доступа к файлам и каталогам. Она позволяет управлять разрешениями на чтение (r), запись (w) и выполнение (x) для владельца, группы и остальных пользователей, используя числовой (например 755) или символьный (u+x) форматы

* chmod 755 file Владелец может все (7), группа и остальные читать и выполнять (5)
* chmod 644 file Владелец только читает и пишет (6), группа и остальные только читают (4)
* chmod 777 file Полные права для всех (чтение, запись, выполнение)

* chmod u+x file Добавить право исполнения (x) владельцу (u)
* chmod g-w file Забрать право записи (w) у группы (g)
* chmod o=r file Установить права для остальных (o) только на чтение (r)

* chmod -R 755 /path/to/dir - Рекурсивное изменение (изменяет права для папки и всех папок и файлов внутри неё)

Символьные категории пользователей:

* u - пользователь (владелец)
* g - группа 
* o - остальные
* a - все пользователи

Права часто отображаются как **rwxr-xr-x**, где первые три символа - владелец, вторые - группа, последние - остальные

							chown:

Команда chown (change owner) используется для изменения владельца и-или группы файла или каталога. Она позволяет передавать права собственности на объекты между пользователями, что требует привилегий суперпользователя. 

Основной синтаксис: chown [опции] новый_владелец[:новая_группа] файл или каталог

* sudo chown username file.txt Смена владельца
* sudo chown username:groupname file.txt Смена владельца и группы
* sudo chown -R username:groupname /path/to/folder Рекурсивная смена (для папок)
* sudo chown :groupname file.txt Смена только группы. Символ : перед именем группы позволяет поменять только группу, оставляя владельца прежним

							top:

Команда top - встроенная утилита мониторинга системы в реальном времени, отображающая список всех активных процессов, использование ЦП, памяти и нагрузку (load average). Она работает как диспетчер задач в текстовом режиме, автоматически обновляя данные (по умолчанию каждые три секунды)

Горячие клавиши:
* z - Цветной режим
* x - Подсветка сортируемого столбца
* с - Показать полный путь к команде
* u - Фильтр по конкретному пользователю*
